# TextFormatter

Algoritmo capaz de formatar um arquivo texto(somente txt), de acordo com os dados informados pelo usuário.

## Getting Started

Para realizar a formatação, será necessário informar:
 * Número de caracteres permitido por linha;
 * Local do arquivo à ser formatado;
 * Local e nome do novo arquivo formatado;

* Para visualizar arquivo formatado [Clique aqui](https://github.com/Rabugari/TextFormatter/blob/master/files/text-formatted.txt)
